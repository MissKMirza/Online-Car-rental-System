# Online-Car-rental-System

<h2> Instructions: How to Run? </h2>
1. After you finish downloading the project, unzip the project file and head over to the project root folder. <br>
2. You can also create a Virtual Environment and activate it.<br>
3. Open your Terminal/Command Prompt in the project’s root folder.<br>
4. Install the Requirements: pip install -r requirements.txt.<br>
5. Requires a MySQL database, so create a database with the name given under the settings.py file.<br>
6. Then, make database migrations: python manage.py makemigrations<br>
7. python manage.py migrate<br>
8. And finally, after a successful migration run, the application: python manage.py runserver<br>
9. At last, open up your favourite web browser<br>
10. Go to URL “http://127.0.0.1/[ PORT_NUMBER ]/“<br>
11. For the Admin Panel credentials, you have to create one with a superuser.<br>
